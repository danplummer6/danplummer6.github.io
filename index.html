AppShare

AppShare was my final project at Full Sail University. Although the website is not live, the code lives on here at GitHub. Appshare was built using the php framework Codeigniter, Bootstrap, MySql, HTML5, CSS3, and JavaScript.

In this project my professor acted as a venture capitalist investing in my idea and I was responsible for presenting a project proposal, performing maintenance after launch, and everything in between. The concept of the website is to have users register and then be able to submit ideas for web / mobile applications, as well as explore other peoples app submissions. Users also have a dashboard where they can submit new ideas and view, edit, and delete all of their submissions. Extra features include form validation, password encryption, 5 star rating system, commenting on submissions, and a facebook share button for each app submission.

Once my concept was approved the project was completed in the following stages.

  1. Logo design and style guide.
  2. Create wireframes and design Comps.
  3. Begin development phase.
  4. Alpha stage.
  5. Beta stage.
  6. Debug and maintenance.
  7. Site Launch and post mortem.
